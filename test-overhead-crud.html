✅ Test Overhead CRUD Master Data System

1. Fitur CRUD

Operasi Cara Kerja API Endpoint
CREATE Click "+ Tambah" di form "Tambah Item Baru" Form input menciptakan elemen baru, Save menyimpan via API
READ Data dimuat saat load halaman dari database GET /api/master-data.php?action=get_overhead
UPDATE Edit nilai di input field, click "Simpan Overhead" POST /api/master-data.php dengan type:'overhead'
DELETE Click tombol ✕ di setiap item (akan auto-save) Item di-remove dari DOM, data disimpan via API

2. Struktur Data

Format di settings.json:

{
  "overhead": {
    "designer": 20000000,
    "marketing": 15000000,
    "creative": 8000000,
    "pm": 8000000,
    "sosmed": 7000000,
    "freelance": 4000000,
    "ops": 12000000
  }
}
💡 Keys bisa apa saja - sistem mendukung item custom

3. JavaScript Functions

Function Tujuan Dipanggil dari
saveOH() Collect semua .overhead-item dan simpan via API "Simpan Overhead" button
addOHItem() Tambah item baru ke DOM (belum ter-save) "+ Tambah" button di form tambah item
deleteOHItem(name) Hapus item dari DOM dan auto-save "✕" button di setiap item
updateOHTotal() Hitung dan tampilkan total overhead Input event pada semua .ov-value fields
resetOH() Reset ke default values "Reset ke Default" button

4. Data Flow

User Flow:
1️⃣ Halaman load → getSettings() → settings.json overhead
2️⃣ PHP: $overhead array di-render ke `.overhead-item` elements
3️⃣ JS: DOMContentLoaded → updateOHTotal() + event listeners
4️⃣ User edit value → input.oninput → updateOHTotal() (real-time)
5️⃣ User click "+ Tambah" → addOHItem() (client-side)
6️⃣ User click "✕" → deleteOHItem() → saveOH() (auto-persist)
7️⃣ User click "Simpan" → saveOH() → updateMasterData() API call
8️⃣ API: POST /api/master-data.php → updateOverhead() → settings.json
9️⃣ Response: success → show status message
🔟 Other pages: call loadMasterDataFromAPI() to get latest overhead

7. API Master-Data Endpoint

GET Requests:

GET /api/master-data.php?action=get_overhead
Response: { "success": true, "data": { "designer": 20000000, ... } }

GET /api/master-data.php?action=get_all
Response: { "success": true, "data": { overhead, cetak_f, grad, ... } }

POST Request (Update):

POST /api/master-data.php
Content-Type: application/json

{
  "type": "overhead",
  "data": {
    "designer": 21000000,
    "marketing": 16000000,
    ...
  }
}

Response: { "success": true, "message": "Master data updated", "data": {...} }

Problem Cause Solution
Item tidak muncul saat load settings.json tidak punya overhead key atau kosong Verify settings.json structure, check file permissions
Total tidak ter-update Event listener tidak ter-attach ke .ov-value inputs Check console log, verify DOMContentLoaded called
Save gagal updateMasterData() API call error Check browser Network tab, verify /api/master-data.php accessible
Delete tidak work saveOH() error setelah item.remove() Check console for errors, verify API connection
